Custom E-Commerce Payment

Ruby on Rails Application Development

Integrated a third party unsupported payment gateway, providing Internet PIN debit payment. The payment page retrieves payment details from Shopify via their API, presents it to the user for verification, and presents a payment form, with a secure PIN pad. If the payment succeeds, the application sends a debit request via the PayLeap API, and updates the order via the Shopify API.

  • Date: August 2014
  • Technologies: Ruby on Rails, Twitter Bootstrap, Shopify API, PayLeap API, E-Commerce, HAML, AJAX

High Traffic AWS Infrastructure

Amazon Web Services Application Deployment

Created infrastructure and supported the deployment of a Ruby on Rails web application, for a high traffic website. Design infrastructure to be resilient, and capable of handling tens of thousands of users per day. The setup utilised AWS load balancing and autoscaling services.

  • Date: August 2014
  • Technologies: AWS, EC2, RDS, S3, Elastic Load Balancer, AutoScaling, Ubuntu, Nginx, Unicorn, Ruby on Rails, Sidekiq, Redis, Elastic Search, PostgreSQL

Amazon Product Lookup Tool

Retrieves product information via Amazon API.

Developed a tool which accepts a spreadsheet of product codes uploaded by the user, queries the Amazon Product API for product details, and presents the information to the user. Information also made available for download.

  • Date: August 2014
  • Technologies: Ruby, Sinatra, SQLite, RedHat Cloud (RHC), Amazon API
